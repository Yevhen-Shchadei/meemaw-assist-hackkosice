# 👵 Meemaw Assist - AT&T Hackathon 2026

## 🎯 The Problem
Millions of elderly people face simple tech problems every day, but existing support is too technical. They don't know what a "router" or "cache" is.

## 💡 Our Solution
An empathetic, voice-first AI assistant that speaks human language. It guides users step-by-step using visual cues and patience, so your grandpa never has to call you for tech support again.

## 🛠 Tech Stack
* **AI Engine:** Google Gemini Pro / LLaMA 3 (Groq)
* **Backend:** Python (FastAPI)
* **Frontend:** HTML/CSS/JS (Voice-to-Text integration)

## 🚀 How to run locally
1. Clone the repo: `git clone https://github.com/your-team/meemaw-assist.git`
2. Add your API keys to a `.env` file.
3. Run backend: `python backend/main.py`
4. Open `frontend/index.html` in your browser.
